About Us

"I Still Don't Care About Cookies" blocks cookie pop-ups effectively and securely. After Avast acquired the original extension, we created a separate, privacy-conscious version. We're constantly improving it to ensure you browse without interruptions, free from corporate ownership.

Contribute on GitHub

The "I Still Don't Care About Cookies" extension is open source and relies on community support. If you want to help us develop and maintain it, visit our GitHub page. You can contribute code, report issues, or improve translations. Your input makes a difference.
